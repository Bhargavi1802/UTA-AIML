Building CNN:¶

  • Define layers.
  • Set optimizer and loss function. (Use Adam optimizer and categorical crossentropy)

A couple of experimented below :

  1. adding multiple layers and then increasing the conv2D,Maxpooling and dropout layers
  2. Using a lesser number of layers

Lessons learnt

  1. An increase in the number of layers had decreased the accuracy and the model also overfit (The code is hence set to rawNBconvert)
  2. Decreasing the number of layers had significantly increased the accuracy

Fitting the model¶

Due to the very low size of the data we considered the following options:

  1. Train and Test using the individual data sets given ,to validate the model at hand ,once confidence is achieved in the model at hand (code in RawNBcode) ,the kernel is restarted . (Train and Test Accuracy is also plotted to see any overfitting for the number of epochs defined)

  2. After step 1 ,we start the model fit again on the data with 100% train and no exposure to Test (avoiding any possibility of bias in data) .We then expose the Test set and make predictions on that set as normal

Business Insights

True Negative (observed=0,predicted=0)

False Positive (observed=0,predicted=1)

True Negative (observed=0,predicted=0)

False Negative(observed=1,predicted=0)

The metric of main interest should in my opinion here be Accuracy,Recall as we do not want to risk cases that are False Negative .We need to ensure to reduce the number of cases that are falsely identified as healthy when they have contracted either COVID or Pneumonia as this implies the spread of the disease ,on contrary a healthy person being identified as unhealthy means he will be on unnecessary usage of medications.

So fundamentally we DONOT want to identify a COVID patient as a normal healthy person ,which would have it's risks of spreading the disease .Hence a combination of Accuracy and Recall,f1 score should be our metrics of focus

True positive and True negative numbers still have a significant impact on decision making ,identifying the disease under the correct category is still of utmost importance,especially when the person has a disease that is contagious
